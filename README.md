# 🛒 eCommerce Web App

A fully responsive and modern **eCommerce Application** built using **React.js**. This project simulates a real-world online shopping experience where users can explore products, view details, and manage their cart.

## 🚀 Features

* 🛍️ Browse products by categories
* 🔍 Search and filter products
* 📄 Product detail page
* 🛒 Add to cart functionality
* ➕ Increase/decrease product quantity
* 💰 Dynamic cart total calculation
* 🔄 State management using Context API
* 📱 Fully responsive design

## 🛠️ Tech Stack

* React.js
* JavaScript (ES6)
* Context API / useReducer
* CSS / Tailwind / Bootstrap
* React Router DOM
